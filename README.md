# About it
`efsite`是一个练习项目，采用NodeJs + Express + Mongodb开发。

# How to run it
 1. Checkout 代码   
 1. 修改config.js文件，配置Mongodb数据库连接     
 1. 运行`node server.js`，默认监听80端口   

# Where to see it
efsite现部署在了  [NAE](http://www.cnodejs.net)  上，地址是 http://efsite.cnodejs.net
